# Financial Analysis Project

## Описание
Анализ финансовых данных 5 компаний из разных секторов за период 2020-2025.

## Компании
- **NVDA** (NVIDIA) — технологии
- **GS** (Goldman Sachs) — финансы
- **PFE** (Pfizer) — здравоохранение
- **XOM** (ExxonMobil) — энергетика
- **KO** (Coca-Cola) — потребительские товары

## Что анализируется
1. Цены закрытия акций
2. Дневная доходность
3. Волатильность
4. Корреляции между акциями
5. Скользящие средние
6. Кумулятивная доходность

## Технологии
- Python 3.x
- Pandas
- NumPy
- Matplotlib
- yfinance

##Ключевые выводы
Компания с наибольшей доходностью - NVIDIA, цена её акций за данный период выросла в 26 раз.
Компания с наименьшей доходностью - Pfizer, цена её акций за данный период упала на 16%.
Наибольшая корреляция наблюдается между акциями компаний Sachs Group и ExxonMobil Holings, наименьшая - между NVIDIA и Pfizer.
Можно заметить сильный спад цен на акции Pfizer, начавшийся осенью 2023 года. Оказалось, что это случилось по 3 причинам:
1. Сильное обесценивание запасов препаратов от COVID-19 в 2023.
2. В октябре 2023 руководство сильно ухудшило финансовые ожидания на весь текущий год.
3. Инвесторы отыграли завершение "ковидной" эры. Государственный сектор резко ограничил закупки препаратов.
Ситуация со стремительным ростом цены акций на NVIDIA тоже связана с мировым трендом. Начиная с января 2024 начал резко расти спрос на чипы для искусственного интеллекта. Крупнейшие корпорации, такие как Microsoft, Meta и Google, начали вкладывать миллиарды долларов на создании ИИ-инфраструктуры.
